# 短视频生成 (short-video-gen)

通用短视频生成工具。描述主题 → 自动生成 4:3 横屏 + 3:4 竖屏两版视频。

## 功能

- 10 种视觉配色（赛博朋克、极简白、暗金、霓虹……）
- 3 种文案风格（一句话暴力、痛点轰炸、颠覆认知）
- 双比例输出：4:3 横屏 (1600×1200) + 3:4 竖屏 (1080×1440)
- 自动 BGM 合成
- 素材优先使用真实截图

## 依赖

- Node.js ≥ 22
- FFmpeg
- HyperFrames（自动下载）

## 使用

```
/short-video-gen 给我介绍一下今天的 GitHub 热门工具
```

首次使用会自动弹出配置向导，之后一直用同一套风格。

## 配置项

| 配置 | 选项 |
|------|------|
| 时长 | 15s / 20s / 30s / 自定义 |
| BGM | 自动合成 / 不加 / 手动提供 MP3 |
| 视觉 | 10 种配色方案 |
| 文案 | 一句话暴力 / 痛点轰炸 / 颠覆认知 |
| 侧重 | 核心亮点 / 功能罗列 |

## 输出

```
D:\视频生成\<project>\
  43-横屏.mp4
  34-竖屏.mp4
```
