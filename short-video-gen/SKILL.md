# 短视频生成

**质量在设计阶段决定，不在渲染后修补。**

## 执行铁律

1. **Stage 2 设计先于 Stage 3 编码** — 没画完设计稿不准写 HTML
2. **每个阶段产出独立文件** — 选题/设计/脚本各自存为 .md
3. **质量自查** — 每阶段结束时 AI 自己审一遍再进入下一阶段

## 快速启动

缺参数问用户，不缺直接进入 Stage 1。

---

## 特殊类型：歌词 MV

**AI 无法听到音频，歌词对轨是最容易出错的环节。必须遵守以下规则：**

1. **LRC 必须从音乐平台官方获取**（网易云/QQ音乐/酷狗），不随机搜网页上的
2. **音频必须从同一平台下载**，确保版本和 LRC 匹配
3. **渲染后用户必须检查对轨**——AI 无法听音频

### 歌词 MV 视觉铁律

**以下任何一条违反 = 打回重做**：

- [ ] **单行显示**：一句歌词 = 一行文字，不做 main+sub 两行副标题格式
- [ ] **位置变化**：相邻两句不能在同一位置。6 种位置轮换：居中/左对齐/右对齐/左上角/右下角/顶部居中
- [ ] **字号动态**：普通句 34-46px，强调句 58-74px，punchline 96-120px。不能全部统一字号
- [ ] **关键词特效**：攻击性词汇(开炮/剥光/帮倒忙) → 发光+抖动，嘲讽词汇(fake/膨胀/马屁/装的) → 抖动，技术词汇(autotune/freestyle/电音) → 发光
- [ ] **段落配色**：intro 黑 → verse1 暗红 → verse2 暗金 → verse3 暗紫 → peak 红爆 → outro 黑
- [ ] **装饰元素**：每段至少 1 种几何装饰（对角线/虚线环/网格/emoji）
- [ ] **Punchline 放大**：高潮句字号 ≥ 96px，带发光 text-shadow
- [ ] **Emoji 点缀**：关键句配合适 emoji（🎭💀🎯💰💥👑🔥）

### 生成方式

歌词行数多（30-50 行），手写 HTML 容易出错。**用 Node 脚本生成 HTML**：
- 输入：LRC 时间戳数组
- 输出：每行一个 `.clip` div，自动分配位置/字号/特效/emoji
- GSAP 只做进入动画（`scale`/`y`/`opacity`），HyperFrames `data-start/duration` 管理可见性

## Stage 2: 设计（最关键）

**没有通过设计自查，不准写 HTML。**

产出 `02-设计.md`，必须包含：

### 2.1 视觉方案总览
- 配色（底色/主色/辅色/强调色/文字色）
- 字体方案（标题/正文/强调的字号字重）
- 整体氛围描述（3 个形容词）

### 2.2 段落设计（歌词MV）/ 场景设计（其他类型）

**歌词MV 必须逐段落设计**：intro / verse / chorus / bridge / outro

每个段落明确：
```
段落：Verse 1 (0:18-0:60)
底色：深红黑 #1a0000
氛围：紧张、攻击性
文字颜色：白色主文字 + 红色关键词
特效：重音时文字抖动(scale 1→1.15→1, 0.1s)、关键词放大变红
排版：双行居中，上行36px副文，下行56px正文
背景：暗红orb + 细密网格
```

**非歌词MV 必须逐场景设计**：布局 + 素材 + 文字 + 动画

## HyperFrames 核心规则

**HyperFrames 管理可见性，GSAP 只做动画。不要用 GSAP 控制元素显示/隐藏。**

1. 每个时序元素必须是 `<div class="clip" data-start="X" data-duration="Y" data-track-index="Z">`
2. `data-start/duration` 控制可见时段——HyperFrames 自动显示/隐藏，不要用 GSAP `opacity 0→1`
3. GSAP 只用于：弹入(`scale`/`y`)、抖动、变色、发光——在元素已经可见的前提下
4. **歌词 MV**：每行歌词 = 一个 `.clip` div，`data-start`=LRC 时间戳，`data-duration`=到下一句的间隔
5. **背景**：每种背景色/装饰 = 一个独立 `.clip` div，按段落时间轴切换
6. BGM = `<audio class="clip" data-start data-duration>`
7. `paused:true` GSAP timeline 注册到 `window.__timelines["main"]`
8. 不用 `Math.random()`/`Date.now()`
